# Requirement Gathering Checklists

## Web Application Checklist
1. **Frontend**: Next.js, Vite, React, or Vanilla?
2. **Styling**: TailwindCSS, CSS Modules, or Vanilla CSS?
3. **State Management**: Redux, Context API, or None?
4. **Backend/API**: Node.js, Python, or Mock data?
5. **Database**: PostgreSQL, MongoDB, or JSON file?
6. **Deployment**: Vercel, Netlify, or Local only?

## System Integration Checklist
1. **Protocols**: CAN, LIN, SPI, or I2C?
2. **Hardware**: specific board or simulation?
3. **Sampling Rate**: Required frequency for data capture?
4. **Data Format**: Hex, Decimal, or Raw binary?
